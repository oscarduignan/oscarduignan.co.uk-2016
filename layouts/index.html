How is this site built?

I write the content for this site in Markdown, which I give to a static site generator called Hugo that converts it into HTML. Then I use a tool called s3-website to compress and push the built site to Amazon S3 where it is hosted. The title font used on the site is called Alike Angular and comes from Google's font service. I manage the CSS, JS, other assets used to create the sites design and functionality using Webpack. You can view the source code for the site at github.com/oscarduignan
